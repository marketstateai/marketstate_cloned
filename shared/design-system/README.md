# Design System

Shared UI primitives, tokens, and component standards.

## Structure

- Root-level MCP server lives in `/mcp`.

## Scope

This area is the canonical source of truth for:

- global brand tokens
- approved theme variants
- reusable component contracts
- layout recipes
- design assets metadata

The MCP server consumes these contracts from the central `/mcp` area so internal tooling has one entrypoint.
